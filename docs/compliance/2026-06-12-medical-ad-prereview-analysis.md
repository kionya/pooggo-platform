# RichDoc 후기 노출의 의료광고 사전심의 성격 규명 (의료법 제56·57조 / 의료해외진출법)

작성일: 2026-06-12
근거: deep-research(현행 2024–2026 법령·자율심의기구 기준·판례, 25개 소스 → 19개 적대검증 확정 주장)
성격: **법률 자문 보조자료** — 최종 판단 아님. ⚠️ 표시 항목은 자율심의기구(대한의사협회 admedical.org)·의료광고 전문 변호사 확인 필수.

## 0. 핵심 결론 (TL;DR)

**RichDoc의 후기 노출은 "로그인 게이팅했으니 안전"이 아니다.** 위험은 후기 자체보다 **플랫폼 구조(유료 등급·건당 수수료·큐레이션)와 한국어 국내 노출**에서 온다.

| 쟁점 | RichDoc 판단 | 위험도 |
|---|---|---|
| 플랫폼이 의료광고 적법 주체? | **불가**(제56조①: 의료인·의료기관만). 위반 주체는 될 수 있음 | 구조적 |
| 건당 수수료+유료등급+추천 구조 | **제27조③ 유인·알선 포섭 위험**(대법 2018도20928) | 🔴 높음 |
| 환자 치료후기 노출 | 효과 오인 우려 시 제56조②2호 금지. **플랫폼 큐레이션+제휴병원 결합 시 광고로 기능** | 🟠 중상 |
| 로그인 게이팅 | **자동 면제 아님**(누구나 가입 가능하면 불특정다수성 유지, 복지부 해석) | 🟠 중 |
| 한국어 국내 노출 | **최대 위험지대** — 외국인유치 국내광고 금지(제56조②12호) a fortiori 포섭 | 🔴 높음 |
| 사전심의 대상 매체 여부 | 일평균 이용자 **10만↑**면 직접 대상. 미달 시 RichDoc 자체는 비대상이나 게시 광고·플랫폼 책임은 별개 | ⚠️ 사실의존 |

→ **현 단계 권고: 한국어 로케일에서 후기·치료성 콘텐츠 노출을 보수적으로 운용**하고, ②(큐레이션 아닌 사후 모더레이션)·③(가입 진입장벽)으로 "큐레이션 아님 + 폐쇄적 게이팅"을 강화하되, **외국인 유치 사업의 국내광고 성격 자체를 자율심의기구·변호사에게 확인**해야 한다.

## 1. 광고 주체성 — 비의료인 플랫폼은 적법 광고 주체가 될 수 없다

- **제56조①**: 의료기관 개설자·의료기관의 장·의료인이 **아닌 자는 의료광고를 할 수 없다.** 제3자(플랫폼)가 대행해도 광고 주체는 의료기관이어야 하고, 제3자는 **장소대여 등 위탁업무로 한정**(복지부 보도자료 2022, list_no=371102). [근거: 3-0 확정 / admedical.org, mohw.go.kr, 법령 제56조]
- **함의**: RichDoc은 적법한 광고 주체가 **될 수 없고**, 게시 콘텐츠가 의료광고로 평가되면 **위반 주체**가 된다. 후기·병원정보 노출이 "플랫폼의 의료광고 행위"로 귀속될 위험.

## 2. 유인·알선(제27조③) — 건당 수수료 구조가 핵심 위험

- **대법원 2018도20928**: 비의료 플랫폼 운영자가 시술 홍보 배너 운영 + 환자 예약 시 **치료비의 일정 비율 수취** → 제27조③ **영리 환자 유인·알선 위반**. [3-0 확정]
- **대법원 2010도1763**(2012.9.13): 30만 회원에게 라식 이벤트 이메일 = **불특정 다수 대상 일반광고로 유인 아님**. [3-0 확정]
- **경계선**: 「수익배분 전제 + 특정 환자 알선·중개」 = 유인 / 「순수 불특정 다수 광고」 = 적법.
- **RichDoc 적용**: 건당 수수료 + 유료 제휴등급(베네핏/제휴) + 추천 구조는 **전자(유인)에 가깝다 → 🔴 고위험**. 단 사실관계 의존. ⚠️ 일부 소스의 "수수료 15–20%" 수치는 환각 가능 — 핵심 판시(비의료 플랫폼의 제27조③ 포섭)만 신뢰.
- ※ **외국인 환자에 한해서는** 의료해외진출법상 유치업자 등록 시 수수료가 합법화(사업 전제, [[richdoc-global-hub-roadmap]]). 따라서 제27조③ 리스크는 **국내 환자에게 노출·유인되는 경우**에 집중된다 → 한국어 국내 노출 문제(§5)와 직결.

## 3. 환자 치료후기 — "효과 오인 우려" + 큐레이션이 위법 트리거

- **제56조②2호 / 시행령 제23조①2호**: 소비자로 하여금 **치료 효과를 오인하게 할 우려**가 있는 환자 치료경험담 광고 금지. 의료인의 후원·비용지원 후기, 특정기관 구체 진료경험·수술예후 광고 = 금지 경험담 광고. [3-0 확정 / admedical.org, mohw]
- **예외 여지**: 요건이 "**효과 오인 우려**"이므로 **모든 자발적·비큐레이션 후기가 자동 금지는 아니다.**
- **RichDoc 적용**: 플랫폼이 후기를 **선별·배열(큐레이션)** 하거나 제휴병원 노출과 결합하면 "광고로 기능"할 위험↑. → **②의 설계 원칙**: 플랫폼이 후기를 **추천·정렬·강조하지 않고**(큐레이션 회피), 효과 오인 표현은 **사후 차단**, 별점 평균을 병원 우열로 강조하지 않는다.
- ※ 이미 구현된 금지어 스캐너(보장/완치/100% 등) + 면책문구는 이 방향과 일치. ②는 여기에 **신고·사후삭제**로 "효과 오인 후기"의 잔존을 차단.

## 4. 사전심의 대상 매체 — 일평균 10만 이용자 기준

- **제57조①4호**: 인터넷 매체(모바일 앱 포함) 의료광고는 사전심의 대상, **구체 범위는 시행령 위임**(조문에 수치 없음). 현행 효력 2025-09-19. [3-0 확정]
- **시행령 제24조**: 사전심의 대상 인터넷매체(제1항4호)·SNS(제2항) = **전년도 말 기준 직전 3개월 일평균 이용자 10만 명 이상**. 인터넷·SNS 동일. 현행(대통령령 제36083호, 2026-02-10 시행). **10만 기준은 매체(플랫폼) 운영자 단위**(서울행정법원 2024구합74779: 개별계정 팔로워 무관). [3-0 확정]
- **RichDoc 적용**: ⚠️ **RichDoc 일평균 이용자가 10만↑이면 직접 사전심의 대상 매체**. 미달이면 RichDoc 자체는 사전심의 대상 매체가 아닐 수 있으나, **(a) 게시된 각 병원광고의 위법성, (b) 플랫폼의 제56조①/제27조③ 책임은 별개로 남는다.** → 10만 미달이라고 안심 금지.
- ⚠️ 10만 수치는 시행령 개정으로 바뀔 수 있음 — 게재 직전 law.go.kr 재확인.

## 5. 한국어 국내 노출 = 최대 위험지대 (외국인유치 국내광고 금지)

- **제56조②12호 / 시행령 제23조①12호**: 제27조③ **외국인환자 유치를 위한 국내광고 금지.** [3-0 확정]
- **외국어·글로벌 콘텐츠라도** 국내 일반공중에 쉽게 노출되게 구성되면 **위법 국내광고로 재성격화**(dailydental). 한국어 국내 노출은 **명백한 국내광고**.
- **RichDoc 적용**: 현재 **ko(한국어) 로케일을 국내에서 그대로 노출** → 외국인유치 콘텐츠의 국내광고 포섭 위험 🔴. ⚠️ 단 12호의 직접 수범자는 의료인/병원이며, 플랫폼 책임은 제56조①+제27조③ 경로 — **12호 단독으로 플랫폼에 귀속시키는 것은 주체귀속 오류**(검증에서 지적됨).
- **의료해외진출법 제15조 광고특례**(중, ⚠️검증필요): 등록 유치의료기관이 **지정장소**(외국인전용판매장·보세판매장·지정면세점·국제항공노선 공항·무역항)에서 **외국어** 의료광고를 **사전심의 후** 가능. 외국어+협소 지정장소 한정 → **국내 한국어 콘텐츠엔 특례 미적용**. ⚠️ 제15조의 지정장소 목록·**온라인/앱 적용 여부**·절차는 1차 텍스트로 미확인 — 확인 필요.
- → **구조적 권고(②③보다 상위)**: 후기·치료성·유치성 콘텐츠를 **외국어 로케일(en/zh/ja)·외국인 대상으로 한정**하고 **한국어 국내 노출과 기술 분리**(언어기반 접근차단/지오펜싱)가 실질적 리스크 회피가 되는지 자율심의기구·변호사 확인.

## 6. 로그인 게이팅의 법적 평가 — 자동 면제 아님

- **복지부 해석**(중, 2-1): 간편로그인(기존 포털ID/임시ID)으로 **불특정 다수가 사실상 열람** 가능하면 여전히 의료광고로 평가. 위반 여부는 (a)효과오인 콘텐츠 (b)제공자 개입/큐레이션 (c)사실상 접근성에 좌우. [행정해석·실무 기반, 분할표결 — 게이팅 폐쇄성 정도에 따라 평가 달라질 수 있음]
- **RichDoc 적용**: 현재 **누구나 즉시 가입(ACTIVE)** → 불특정다수성 유지 가능성. → **③의 정당화 근거**: 이메일 인증 등 **진입장벽으로 게이팅 폐쇄성을 높이면** 불특정다수성·접근성 평가에서 유리. 다만 "회원만 열람"이 그 자체로 면제는 아님.

## 7. 자율심의기구 (2018 사전심의 부활 이후)

- 헌재 2015헌바75 위헌 → 자율심의기구 도입 → 2018 사전심의 부활. 현행 복지부 인정 자율심의기구(**대한의사협회 의료광고심의위원회 등, admedical.org**)가 10만↑ 인터넷매체/SNS 의료광고를 심의. [admedical.org 1차]
- 실집행상 **후기/경험담이 주된 위반유형**(2022 모니터링 286건 중 245건=85.7%가 비의료인 통한 경험담/시술광고; 2024 366건 중 위장후기 31.7%). [3-0 통계]

## 8. 권고 액션

### 즉시(코드 반영 가능 — ②③와 연계)
1. **큐레이션 회피**(§3): 플랫폼이 후기를 추천·강조·정렬하지 않음. 시간순·중립 노출만. 별점 평균을 병원 우열 비교로 강조 금지. → ② 설계 원칙에 반영.
2. **효과 오인 후기 사후 차단**(§3): 신고·관리자 숨김/삭제 + 기존 금지어 스캐너 보강. → ② 구현.
3. **가입 진입장벽**(§6): 이메일 인증으로 게이팅 폐쇄성↑. → ③ 구현.
4. **면책 상시 노출 유지**(이미 구현): 후기 = 개인경험, 효과 비보장.

### 구조적 검토(코드만으로 해결 불가 — 사업·법률 결정)
5. **한국어 국내 노출 분리**(§5, 🔴): 후기·치료성·유치성 콘텐츠를 외국어 로케일/외국인 대상으로 한정하고 한국어 국내 노출과 분리할지 결정. **가장 큰 리스크 레버.**
6. **수수료·등급 구조의 제27조③ 적합성**(§2, 🔴): 건당 수수료+유료등급이 국내 환자에게 유인으로 작동하지 않도록 외국인 한정 구조 점검.

### ⚠️ 자율심의기구·전문 변호사 확인 포인트 (게재·확대 전 필수)
- (A) RichDoc 일평균 이용자 10만 도달 여부 = **직접 사전심의 대상 매체** 여부(§4).
- (B) 후기 큐레이션+등급+게시가 구조가 **적법 불특정다수 광고(2010도1763) vs 수익배분 유인·알선(2018도20928)** 중 어디인지(§2). 건당 수수료 구조의 제27조③ 해당성.
- (C) **의료해외진출법 제15조 외국어 광고특례의 온라인/앱 적용 여부**, 언어기반/지오펜싱 분리가 제56조②12호 리스크를 실질 회피하는지(§5).
- (D) 외국인유치 플랫폼 외국어 후기에 대한 자율심의기구 실제 심의 절차·기준, 미심의 게재 제재수준(과태료/형사)·플랫폼vs병원 책임배분 최신 동향(§7).
- (E) 외국인환자 유치업자 등록 요건·범위 재확인([[phase4-compliance-followups]] #2).

## 9. 인용 금지(검증에서 기각)
- "10만 미만 앱은 사전심의 사각지대로 제외" 주장(0-0/1-0 기각).
- "사전심의 대상 매체가 한정 열거된다" 주장(1-2 기각).
- 대법 2018도20928의 "광고 vs 유인 일반론 구분" 정식화(1-2 기각) — 핵심 판시(비의료 플랫폼 제27조③ 포섭)만 신뢰.
